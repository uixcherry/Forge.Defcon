# Forge Defcon

PC scanner for detecting cheats, prohibited software, and DMA hardware. Built with WPF (.NET 9).

## Документация для администраторов

Полное руководство по детекторам и выявлению читеров: [Forge.Defcon/README.md](Forge.Defcon/README.md)

## Быстрый старт

```bash
cd Forge.Defcon
dotnet restore
dotnet build
dotnet run
```

Требуется **Windows** и **.NET 9 SDK**.
